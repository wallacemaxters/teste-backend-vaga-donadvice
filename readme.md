# Teste Backend DONADVICE

- Faça o clone do projeto
- Rode `composer install`
- Copie o arquivo `.env.example` para `.env` e edite as configurações para o banco de dados.

- Rode o commando `php artisan migrate`. Caso deseje semear os dados iniciais rode `php artisan migrate --seed --seeder=MockSeeder`. Se `MockSeeder` não for encontrado, rode `composer dump` e rode o comando anterior novamente.


- Rode `php artisan key:generate`.

- Rode `php artisan serve` para ver os dados


Acesse a documentação [aqui](https://documenter.getpostman.com/view/245362/TVYKbwpf)
